# StopwatchApp (VB.NET)
Aplikasi Stopwatch sederhana (WinForms) untuk tugas kuliah.
Fitur:
- Start / Stop / Reset
- Lap (daftar waktu)
- Export ke CSV
- (Opsional) Simpan lap ke SQLite

## Requirement
- Windows
- Visual Studio (dengan .NET desktop workload)

## Cara build & run
1. Buka `StopwatchApp.sln` di Visual Studio
2. Build -> Run (F5)
